API使用说明

1. 选择API服务商

请选择您想要使用的AI服务商，我们支持Deepseek、火山引擎、OpenAI和硅基流动等主流服务商。

2. 配置API密钥

请输入您从API服务商处获取的API密钥。API密钥是访问AI服务的凭证，请妥善保管。

3. 选择模型

请选择您想要使用的AI模型。不同的模型有不同的特点和性能，您可以根据自己的需求选择。

4. 调整参数

您可以调整创意度和最大生成长度等参数，以获得更符合您期望的生成结果。

5. 测试连接

点击"测试连接"按钮，系统会测试API连接是否正常，并生成一个示例文本。

6. 保存配置

点击"保存配置"按钮，系统会保存您的API配置，并在游戏中使用该配置生成剧情。

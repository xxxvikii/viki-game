# 吾家有女初长成 - 古风高自由度随机文字人生模拟器

一款基于古代/民国初期背景的文字人生模拟网页游戏，以五位女儿中任意一位为第一视角，沉浸式体验从12岁到寿终正寝的完整人生。

## 游戏特色

- **高自由度抉择**：每一个选择都会影响人生走向，无强制剧情
- **剧情随机性**：每次游戏都有不同的人生体验
- **NPC"活人感"**：NPC具备自主社交、独立人生轨迹、情感动态变化
- **AI驱动剧情**：支持多种AI服务商API接入，生成丰富多样的剧情内容
- **完整人生体验**：从12岁到寿终正寝，体验完整的人生历程
- **温馨喜剧风格**：主打温馨家庭日常，姐妹互动，轻松愉快的游戏体验

## 核心玩法

- **家庭日常**：体验古代家庭生活，与家人互动
- **姐妹互动**：与四位姐妹相处，建立深厚感情
- **技能养成**：学习各种古代技能，提升自身素养
- **婚恋选择**：经历古代婚恋过程，选择自己的人生伴侣
- **社交圈层**：与各种NPC互动，扩展社交圈
- **家族兴衰**：见证家族的发展与变化

## 开始游戏

1. 打开 `index.html` 文件进入游戏主页
2. 点击"开始游戏"按钮创建新角色
3. 随机生成家族设定和角色信息，或重新随机直到满意
4. 确认进入游戏，开始你的古代人生之旅

## API配置

游戏支持多种AI服务商API接入，配置步骤：

1. 点击游戏主页的"API接口配置"按钮
2. 选择API服务商（如Deepseek、火山、openai、硅基流动）
3. 输入API密钥
4. 点击"保存配置"按钮
5. 点击"测试连接"按钮验证API是否可用

## 游戏操作指南

- **推进时间**：点击底部操作栏的"推进下一年"按钮
- **查看信息**：点击左侧信息栏查看个人信息和技能
- **互动事件**：在中间核心内容区选择事件进行互动
- **人物关系**：点击底部操作栏的"查看人物关系"按钮
- **存档读档**：点击底部操作栏的"存档/读档"按钮

## 注意事项

- 游戏需要浏览器支持JavaScript
- API配置需要有效的API密钥
- 存档数据保存在浏览器本地，清除浏览器数据会导致存档丢失

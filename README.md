# 吾家有女初长成

## 项目简介

《吾家有女初长成》是一款由AI驱动的古风高自由度随机文字人生模拟器网页游戏。游戏以古代/民国初期为背景，玩家将扮演一个12岁的少女，体验从12岁到寿终正寝的完整人生旅程。

### 核心特色

- **高自由度抉择**：每个选择都会影响角色的人生轨迹，没有固定的剧情路线。
- **剧情随机性**：每次游戏都会生成不同的事件和人物，保证游戏体验的多样性。
- **NPC"活人感"**：NPC具有独立的性格、行为逻辑和人生轨迹，会根据玩家的互动做出相应的反应。
- **AI驱动剧情生成**：利用先进的AI技术，生成丰富多样的剧情内容，确保每次游戏都有新鲜感。
- **温馨喜剧风格**：游戏整体风格以温馨喜剧为主，兼顾人生真实起伏。

## 技术栈

- HTML5
- CSS3 (Tailwind CSS)
- JavaScript

## 游戏玩法

### 角色创建

玩家可以选择随机生成家族设定（朝代、年号、姓氏、身份、资产、家风）、主角信息（排行、姓名/闺名、容貌、性格、初始技能、隐性属性）和核心NPC信息（父母、姐妹）。如果对生成的结果不满意，可以点击"重新随机"按钮生成新的角色信息。

### 游戏流程

1. **回合推进**：每回合对应1年，玩家需要处理年度核心事件和随机触发的人物互动事件。
2. **技能养成**：通过各种事件和选择，提升角色的各项技能（女红、烹饪、诗词、书画、音律、礼仪）。
3. **社交互动**：与家庭成员和其他NPC互动，建立关系，影响他们的人生轨迹。
4. **人生选择**：面对各种事件和机遇，做出符合角色性格和目标的选择。
5. **家族发展**：关注家族的整体发展，包括家族成员的成长、婚嫁、事业等。
6. **结局生成**：当角色寿终正寝（50-90岁随机）后，游戏会生成第一视角人生自传，记录家族成员后世情况与家族兴衰。

### API配置

游戏需要配置AI服务提供商的API才能体验完整的游戏功能。支持的API提供商包括：

- DeepSeek
- 火山引擎
- OpenAI
- 硅基流动
- 自定义API

玩家需要在API配置页面输入API密钥和其他必要的配置信息，并测试连接是否成功。

## 页面结构

- **index.html**：游戏主页，展示游戏名称、封面和核心数据面板，提供开始游戏和读档按钮。
- **create-role.html**：角色创建/随机生成页，展示家族设定、主角信息和核心NPC信息，支持重新随机或确认进入游戏。
- **game.html**：游戏主界面，包含时间面板、信息栏、核心内容区、功能区和操作栏。
- **api-config.html**：API配置页，提供API服务商的配置选项，支持API密钥输入、配置保存和验证功能。
- **instructions.html**：游戏说明页，展示游戏核心设定、玩法规则、界面操作指南、NPC互动逻辑等内容。

## 安装与运行

1. 克隆或下载本项目到本地。
2. 使用现代浏览器（如Chrome、Firefox、Edge等）打开index.html文件。
3. 按照游戏指引进行操作。

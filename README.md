# 吾家有女初长成 - 古风高自由度随机文字人生模拟器

## 游戏简介

《吾家有女初长成》是一款基于古代/民国初期背景的文字人生模拟网页游戏。玩家将以五位女儿中任意一位为第一视角，沉浸式体验从12岁到寿终正寝的完整人生，涵盖家庭日常、姐妹互动、技能养成、婚恋选择、社交圈层互动等核心玩法。

### 核心特色

- **高自由度抉择**：每一个选择都会影响人生走向，没有固定剧情
- **剧情随机性**：每次游戏都会生成不同的家族设定、角色信息和事件
- **NPC"活人感"**：NPC具有独立人格、情感变化和人生轨迹
- **AI驱动剧情**：使用AI生成丰富多样的剧情内容，每次游戏体验都不同
- **存档读档**：支持三个存档位，随时保存和加载游戏进度
- **响应式设计**：适配不同设备，提供良好的游戏体验

## 技术栈

## 游戏玩法

### 开始游戏

1. 打开 `index.html` 文件进入游戏主页
2. 点击"开始游戏"按钮
3. 在角色创建页面查看随机生成的家族设定和角色信息
4. 可以点击"重新随机"按钮生成新的设定，或点击"确认进入游戏"开始游戏

### 游戏主界面

游戏主界面分为以下几个区域：

- **顶部时间面板**：显示当前年号、年份、季节和主角年龄
- **左侧信息栏**：显示个人信息和技能面板
- **中间核心内容区**：显示年度事件、人物互动、家庭动态和我的手记
- **右侧功能区**：显示新人物登场、事件记录、书信箱和家宅地图
- **底部操作栏**：提供推进时间、查看关系、存档读档等功能

### 核心玩法

1. **年度事件**：每个季节会触发3-5个核心事件，玩家需要做出选择
2. **人物互动**：与家人和其他NPC互动，提升好感度
3. **技能养成**：通过学习和实践提升各种技能
4. **婚恋选择**：在适婚年龄选择合适的婚恋对象
5. **家族经营**：管理家庭资产，维护家族关系
6. **人生结局**：寿终正寝后生成人生自传，总结一生经历

## API配置

游戏支持通过API生成剧情内容，提高游戏的丰富度和随机性。

### 配置方法

1. 在游戏主页点击"API接口配置"按钮
2. 选择API服务商（Deepseek、火山、OpenAI、硅基流动或自定义）
3. 输入API密钥和其他配置信息
4. 点击"测试连接"按钮验证API是否可用
5. 点击"保存配置"按钮保存设置

### 支持的API服务商

- Deepseek（推荐）
- 火山
- OpenAI
- 硅基流动
- 自定义模型

### 离线模式

如果没有可用的API，游戏会自动切换到离线模式，使用预设的剧情内容。

## 存档系统

游戏支持三个存档位，可以随时保存和加载游戏进度。

### 保存游戏

1. 在游戏主界面点击底部操作栏的"存档"按钮
2. 选择一个存档槽
3. 点击"保存"按钮

### 加载游戏

1. 在游戏主页点击"读档"按钮
2. 选择一个存档槽
3. 点击"加载"按钮

### 导入导出存档

1. 在存档管理界面可以导出存档文件
2. 也可以导入之前导出的存档文件

## 游戏说明

游戏包含详细的游戏说明页面，可以在游戏主页点击"游戏介绍"按钮查看。

说明内容包括：
- 核心设定

- NPC互动

- 婚恋系统

- 常见问题
